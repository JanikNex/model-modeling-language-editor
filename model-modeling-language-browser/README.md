## Get started

Run

```bash
npm i
npm run serve
```

and open http://localhost:3000 in your browser.

The **base** directory contains the exported **mmlServerWorker** generated from the
official [Model-Modeling-Language](https://github.com/JanikNex/model-modeling-language) repository.
